# Botium Toys Security Audit

This project contains a cybersecurity internal audit for the fictional company Botium Toys.

The objective of this audit is to evaluate the organization's security controls,
risk posture, and compliance practices.

## Framework used

NIST Cybersecurity Framework

## Files

- Botium risk assessment report
- Control categories documentation
- Completed controls and compliance checklist 

## Key findings

The audit identified several missing security controls, including:

- Lack of encryption for sensitive data
- No intrusion detection system (IDS)
- No disaster recovery plan
- No data backups
- Weak password management practices

## Recommendations

- Implement encryption for sensitive customer data
- Deploy an intrusion detection system
- Establish a disaster recovery plan
- Implement backup procedures
- Improve access control policies

- Risk assessment
